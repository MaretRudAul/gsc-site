# GSC Travel Grant

The GSC awards monthly travel grants to graduate students attending academic conferences, workshops, or professional development events. Awards are distributed by lottery from eligible applicants.

**Award period:** August–May (10 awards per academic year · no awards in June/July)

## How to apply

:::info Step 1 — one-time setup
The application form changes each academic year. Find the current form at [studentorg.vanderbilt.edu/gsc/travel-funding-request](https://studentorg.vanderbilt.edu/gsc/travel-funding-request/).
:::

1. Complete the application form before the monthly deadline
2. A lottery is run from all submitted names
3. If your name is drawn, the academic chairs verify your eligibility
4. Eligible winners receive an email from [gsc_academic@vanderbilt.edu](mailto:gsc_academic@vanderbilt.edu) with next steps
5. If the first winner is ineligible, the next name is drawn

## Eligibility

- Currently enrolled in a Graduate School program
- Have not received a GSC Travel Grant recently (check with the academic chairs if unsure)

## Questions?

Email [gsc_academic@vanderbilt.edu](mailto:gsc_academic@vanderbilt.edu)

[Apply for the Travel Grant →](https://studentorg.vanderbilt.edu/gsc/travel-funding-request/)

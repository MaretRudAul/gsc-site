# Graduate Student Spaces

The GSC uses and helps coordinate graduate-student-facing spaces in Alumni Hall. In particular, the transition document notes that the GSC President is granted the ability to reserve Alumni Hall 100 (the Grad Lounge) and that Anna Thomas in the Graduate School is the main logistics contact for Graduate School support related to GSC space access and event coordination.

## Need to use Alumni Hall 100?
If you need to use Alumni Hall 100, contact Anna Thomas in the Graduate School.

## Planning an event?
Campus event space is highly competitive. Major recurring events such as GBMs, socials, and 3MT should be scheduled in the spring for the following academic year. In most cases, event reservations should be paired with an AnchorLink event.

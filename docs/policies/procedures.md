# GSC Procedures

This page covers common procedures for graduate students and GSC members.

---

## How to email the graduate student body

Only the President, VPs, and Secretary can send emails to all ~2,400 Graduate School students via AnchorLink.

1. Log into [AnchorLink](https://anchorlink.vanderbilt.edu)
2. Go to **My Organizations → Graduate Student Council → Roster**
3. Click **Messaging → Create message relay** (the blue + icon)
4. Set the subject and select "Member" as the recipient position
5. Copy the relay email address (looks like `xxxxxxx@relay.engage.campuslabs.com`)
6. Compose your email in Outlook and paste this address as the recipient
7. **Note:** Relay links expire after 24 hours — generate a new one if needed

---

## How to make an event on AnchorLink

1. Log into AnchorLink → My Organizations → Graduate Student Council
2. Click **Events → Create Event**
3. Fill out the form and submit
4. You'll receive email confirmation when approved

All executive board members can create AnchorLink events.

---

## How to reserve a space on campus

1. Log in to [Vanderbilt Virtual EMS](https://25live.collegenet.com/pro/vanderbilt)
2. Click **Reservations** and select the space type
3. Enter date, time, and estimated attendance
4. Select the room and fill in required fields (use GSC as the organization)
5. Submit — you'll be contacted when approved

In almost every case, also create an AnchorLink event for the reservation.

**Note:** Book early. Event space is extremely competitive at Vanderbilt, especially SLC ballrooms. Major recurring events (GBMs, socials, 3MT) should be booked in the spring for the following year.

:::tip Alumni Hall 100 (Grad Lounge)
The GSC President is granted the ability to reserve Alumni Hall 100. If you need this space, contact Anna Thomas in the Graduate School.
:::

## How to make purchases

:::info Allow 2–3 weeks
The purchasing approval process takes time. Start early — especially for events.
:::

1. Make sure you have a space reservation and AnchorLink event (if for an event)
2. Determine exactly what you want to buy and compile costs (make PDFs of carts/quotes)
3. Email the Faculty Advisor and Treasurer (cc the President) requesting approval with item details and costs
4. Fill out the [Purchasing Request form on AnchorLink](https://anchorlink.vanderbilt.edu/organization/GSC)

Key account codes (for purchase requests):
- **Project #:** SO_280200
- **Task #:** 57824
- **Expenditure org:** 30500

Full purchasing reference is in the Treasurer's folder in the [GSC Box](https://vanderbilt.box.com/s/crffhvfojzjvgwv7jia5n0gwzyzyjd97).

## How to check the GSC account balance

1. Log into AnchorLink → My Organizations → Graduate Student Council
2. Click **Finance → Accounts → Graduate Student Council**
3. Click **Transactions** to view activity

Access: President, VPs, Treasurer, Secretary.
